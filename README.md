# ESXDesafio 

Web API Rest de gerenciamento de patrimônios de uma empresa

## Requisitos técnicos

Asp.Net Core, Entity Framework Core, InMemoryDatabase e SQLServer.

##

### Diretórios

- src - Pasta com os fontes do projeto
- doc - Pasta com os testes postman da Api


### Setup

- InMemory
	Executar o projeto no IIS Express, ao executar pela primeira vez, o projeto executa a classe Seed para gerar e popular a base com dados iniciais.
